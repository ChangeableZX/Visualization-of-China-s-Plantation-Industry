# 中国种植业数据可视化 (Visualization of China's Plantation Industry)

本项目是一个基于 Web 技术的数据可视化交互页面，旨在直观展示中国种植业的相关数据分布与演变趋势。

## 🌐 在线预览
您可以直接点击下方链接查看交互效果：
👉 [点击此处访问在线演示页面](https://changeablezx.github.io/Visualization-of-China-s-Plantation-Industry/)

## 🚀 项目功能
- **动态图表展示**：使用 JavaScript 图表库（如 ECharts/D3.js）呈现复杂的农业数据。
- **多维度筛选**：支持通过交互式 UI 切换不同的种植作物或年份。
- **响应式设计**：适配不同尺寸的屏幕，确保在移动端和网页端均有良好的视觉体验。

## 🛠️ 技术栈
- **前端核心**：HTML5, CSS3, JavaScript
- **数据处理**：JSON/CSV 数据解析
- **可视化库**：ECharts / D3.js (根据你实际使用的库修改)
- **托管平台**：GitHub Pages
